So stdio.h means standard I/O, so there’s printf to output stuff and scanf to receive them.

So let’s look at scanf. First you need a string that says what type of variable to look for “%d” stands for digit, the second parameter “&x” means where to store the variable, look that we used the & because we want to send the direction not the value.

#include <stdio.h>

int main()
{
    int a = 10;

    printf("Write 10 or 20: ");
    scanf("%d", &a);

    switch ( a ) {
        case 10:
            printf("Well done!");
            break;
        case 20:
            printf("You're really good");
            break;
        default:
            printf("Are you serious?");
            break;
    }
    
    return 0;
    
}
